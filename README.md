# ModuPrompt - Modular Prompt Builder

A web-based application for creating and organizing text prompts with two different editing modes:

1. **Jupyter Lab Notebook Style** – A cell-based interface for stacking, annotating, and reordering content.
2. **Node-Based Editing** – A visual flow interface where nodes connect via inputs and outputs.

## Features

- Markdown snippet management with folders and subfolders
- Frontmatter (YAML) support for metadata
- Drag & drop insertion
- Markdown preview with Mermaid diagrams support
- Export to Markdown and PDF
- Cell/Node-specific formatters
- Collapsible sections
- And more!

## Development Status

This project is currently in early development.
